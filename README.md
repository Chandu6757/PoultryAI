# PoultryAI 🐔

## AI-Based Poultry Disease Detection using EfficientNet and YOLO

### Technologies Used
- Python
- TensorFlow
- Streamlit
- OpenCV
- EfficientNet
- YOLOv8

### Features
- Detects poultry diseases from images
- Classifies healthy and infected chickens
- User-friendly Streamlit interface
- AI-powered prediction
